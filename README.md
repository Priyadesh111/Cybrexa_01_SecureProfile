
# SecureProfile - Personal Portfolio Website

## Overview

SecureProfile is a personal cybersecurity portfolio website built using HTML, CSS, and JavaScript.

The website showcases my background, skills, education, resume, and contact information as an aspiring SOC Analyst and Cybersecurity student.

---

## Features

- Responsive Design
- Black and Purple Cybersecurity Theme
- Hamburger Navigation Menu
- About Me Section
- Skills Showcase
- Resume Download/View Button
- LinkedIn and GitHub Links
- Contact Form using EmailJS
- Mobile Friendly Layout

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- EmailJS

---

## Pages

### Home
Introduction and portfolio overview.

### About
Professional summary, education, and cybersecurity interests.

### Skills
Core cybersecurity and technical skills.

### Contact
EmailJS-powered contact form.
